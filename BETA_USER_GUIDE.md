## Build Your First App (3 minutes)

### Step 1: Navigate to Builder
- Click **"Build"** in the left sidebar
- Or click **"Build New App"** from Dashboard

### Step 2: Describe Your App
Enter a simple prompt. Try one of these:

**Pharma Examples:**
```
batch tracking system for GMP compliance
deviation management dashboard with CAPA tracking
sample testing workflow for QC lab
```

**Biotech Examples:**
```
fermentor monitoring dashboard with real-time sensors
cell culture process tracker
bioreactor control panel
```

**General:**
```
project task tracker with status updates
inventory management system
customer feedback dashboard
```

### Step 3: Generate
1. **Click** "Generate App"
2. **Wait** 7-10 seconds (you'll see a spinner)
3. **Preview** appears on the right side

### Step 4: Export (Optional)
1. **Click** "Export BASE44" (green button)
2. **Downloads** a JSON manifest
3. **Use it** for regulatory handoffs, platform imports, or documentation
